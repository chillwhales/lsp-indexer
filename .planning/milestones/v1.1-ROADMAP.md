## Overview

Shipped 4 publishable packages (`@lsp-indexer/types`, `@lsp-indexer/node`, `@lsp-indexer/react`, `@lsp-indexer/next`) that give any app type-safe access to all 12 indexer query domains — with client-side (TanStack Query) hooks, real-time WebSocket subscriptions (graphql-ws), and Next.js server actions (`'use server'`). The phases followed vertical-slice delivery: scaffold the package and validate exports → build one domain end-to-end → replicate across all domains → add subscriptions → add server actions → migrate to shared packages → clean up v1 → release prep → CI/CD.

**Package architecture:**

```
@lsp-indexer/types  — Zod schemas + inferred TS types (zero framework deps)
@lsp-indexer/node   — services, parsers, documents, codegen, query keys, execute, errors
@lsp-indexer/react  — thin TanStack Query hooks (browser → Hasura directly)
@lsp-indexer/next   — server actions + hooks routing through them (browser → server → Hasura)
```

## Milestone Summary

**Decimal Phases:**

- Phase 9.12: Block-Ordered Sorting (inserted after Phase 9.11 for deterministic event ordering — 4 event domains needed blockNumber → transactionIndex → logIndex instead of timestamp)

**Key Decisions:**

- 4-package architecture: types/node/react/next with no re-exports across packages
- Vertical-slice approach: build profiles end-to-end, then replicate across 10 domains
- Prisma-style conditional include type narrowing via `IncludeResult` utility type
- 3-overload generic `<const I>` pattern for full type propagation from parser to consumer
- `graphql-ws` for WebSocket subscriptions with TanStack Query cache integration
- Replaced local packages with `@chillwhales/erc725` and `@chillwhales/lsp1` npm packages
- Shared CI/CD infra in `chillwhales/.github` org repo with reusable workflows
- Changesets fixed group for synchronized versioning across all 4 packages

**Issues Resolved:**

- All v1 indexer code removed and v2 promoted to canonical `packages/indexer/`
- All dead/stale comments removed from publishable packages
- JSDoc added to all exported symbols
- CI/CD pipeline fully automated (lint, typecheck, build, test, publish)

**Issues Deferred:**

- Production cutover procedure with rollback plan (deferred from v1.0)
- SSR hydration examples and documentation (SSR-01, deferred to v1.2+)
- Domain-specific stale time tuning (SSR-03, deferred to v1.2+)

**Technical Debt Incurred:**

- 13 requirements have VERIFICATION passed but missing SUMMARY frontmatter (format adopted after those phases completed)
