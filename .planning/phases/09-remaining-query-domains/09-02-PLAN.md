    **packages/react/src/hooks/digital-assets.ts:**
    const DEFAULT_PAGE_SIZE = 20;
    `useDigitalAsset({ address })`: useQuery wrapping fetchDigitalAsset, enabled: Boolean(address), returns { digitalAsset: data ?? null, ...rest }
    `useDigitalAssets(params = {})`: useQuery wrapping fetchDigitalAssets, returns { digitalAssets, totalCount, ...rest }
    `useDigitalAssetSearch(query: string, params = {})`: convenience wrapper — calls useDigitalAssets with filter.name = query merged into params.filter. This is NOT a separate hook implementation; it passes { ...params, filter: { ...params.filter, name: query } } to useDigitalAssets.
    `useInfiniteDigitalAssets(params = {})`: useInfiniteQuery with offset-based pagination, DEFAULT_PAGE_SIZE=20, same pattern as useInfiniteProfiles. Returns { digitalAssets (flattened), hasNextPage, fetchNextPage, isFetchingNextPage, ...rest }.

    **packages/next/src/actions/digital-assets.ts:**
    'use server' directive at top.
    `getDigitalAsset(address: string)`: calls fetchDigitalAsset(getServerUrl(), { address })
    `getDigitalAssets(params?)`: calls fetchDigitalAssets(getServerUrl(), params ?? {})

    **packages/next/src/hooks/digital-assets.ts:**
