    **packages/types/src/owned-assets.ts:**
    ```ts
    OwnedAssetSchema = z.object({
      ownerAddress: z.string(),     // UP address that owns the token
      assetAddress: z.string(),     // token contract address
      balance: z.string().nullable(), // token balance as string (large number)
      name: z.string().nullable(),  // LSP4 token name
      symbol: z.string().nullable(),
    })

    OwnedTokenSchema = z.object({
      ownerAddress: z.string(),     // UP address that owns the NFT
      assetAddress: z.string(),     // collection contract address
      tokenId: z.string(),          // specific token ID
      name: z.string().nullable(),  // token name (LSP4 for this specific token)
      symbol: z.string().nullable(),
    })
    ```
    Filter schemas:
    - `OwnedAssetFilterSchema`: ownerAddress (string), assetAddress (string)
    - `OwnedTokenFilterSchema`: ownerAddress (string), assetAddress (string), tokenId (string)
    Sort: `OwnedAssetSortFieldSchema`: z.enum(['assetAddress', 'balance']), `OwnedTokenSortFieldSchema`: z.enum(['assetAddress', 'tokenId'])
    Hook param schemas for both: UseOwnedAssetsParams, UseInfiniteOwnedAssetsParams, UseOwnedTokensParams, UseInfiniteOwnedTokensParams.
    Export all schemas + inferred types.

    **packages/node/src/documents/owned-assets.ts:**
    `GetOwnedAssetsDocument`: query `owned_asset` with $where (Owned_Asset_Bool_Exp), $order_by, $limit, $offset + aggregate for count.
    Fields: owner_address, address (asset contract), balance { value }, digital_asset { lsp4TokenName { value }, lsp4TokenSymbol { value } }
    `GetOwnedTokensDocument`: query `owned_token` with similar structure.
    Fields: owner_address, address (collection contract), token_id, nft { lsp4TokenName { value }, lsp4TokenSymbol { value } }

    **packages/node/src/parsers/owned-assets.ts:**
