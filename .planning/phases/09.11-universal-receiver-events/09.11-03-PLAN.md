**Create `packages/react/src/hooks/universal-receiver-events.ts`** â€” follows issued-assets.ts hooks pattern exactly (list + infinite, no singular). 2 hooks with 3-overload `<const I>` generic pattern.

**Hooks to create:**

1. **`useUniversalReceiverEvents`** â€” paginated list hook

   - Imports: `UseUniversalReceiverEventsParams`, `UniversalReceiverEvent`, `UniversalReceiverEventInclude`, `UniversalReceiverEventResult`, `PartialUniversalReceiverEvent` from `@lsp-indexer/types`
   - Imports: `fetchUniversalReceiverEvents`, `universalReceiverEventKeys`, `getClientUrl`, type `FetchUniversalReceiverEventsResult` from `@lsp-indexer/node`
   - Uses `useQuery` from `@tanstack/react-query`
   - Query key: `universalReceiverEventKeys.list(filter, sort, limit, offset, include)`
   - Query fn: `fetchUniversalReceiverEvents(getClientUrl(), { filter, sort, limit, offset, include })`
   - Select: extracts `universalReceiverEvents` array from result
   - Returns: `{ data, error, isLoading, isPending, ... }` with `totalCount` extracted from query data
   - Return type alias: `UseUniversalReceiverEventsReturn<F>` using `UseQueryResult<F[], Error>` with `totalCount: number | undefined`

2. **`useInfiniteUniversalReceiverEvents`** â€” infinite scroll hook
   - Uses `useInfiniteQuery` from `@tanstack/react-query`
   - Query key: `universalReceiverEventKeys.infinite(filter, sort, include)`
   - Query fn: passes `offset: pageParam` to `fetchUniversalReceiverEvents`
   - `getNextPageParam`: checks if `totalCount > offset + items.length`
   - Select: flattens pages into single array
   - Returns: `{ data, error, fetchNextPage, hasNextPage, isFetchingNextPage, ... }` with `totalCount`
   - Return type alias: `UseInfiniteUniversalReceiverEventsReturn<F>` using `UseInfiniteQueryResult<InfiniteData<F[]>, Error>`
   - **CRITICAL:** Destructure `fetchNextPage`, `hasNextPage`, `isFetchingNextPage` before rest spread to avoid TS2783 duplicate property errors (established pattern).

**3-overload pattern for both hooks (DX-05):**

```typescript
// Overload 1: No include â†’ full type
export function useUniversalReceiverEvents(
  params?: UseUniversalReceiverEventsParams,
): UseUniversalReceiverEventsReturn<UniversalReceiverEvent>;

// Overload 2: Generic include â†’ narrowed type
export function useUniversalReceiverEvents<const I extends UniversalReceiverEventInclude>(
  params: UseUniversalReceiverEventsParams & { include: I },
): UseUniversalReceiverEventsReturn<UniversalReceiverEventResult<I>>;

// Overload 3: Partial include â†’ partial type
export function useUniversalReceiverEvents(
  params?: UseUniversalReceiverEventsParams,
): UseUniversalReceiverEventsReturn<PartialUniversalReceiverEvent>;
```

Same 3-overload pattern for `useInfiniteUniversalReceiverEvents`.
