**Hook 1: `useDigitalAsset(params: UseDigitalAssetParams)`**

- `queryKey: digitalAssetKeys.detail(params.address, params.include)`
- `queryFn: () => fetchDigitalAsset(url, { address: params.address, include: params.include })`
- `enabled: Boolean(params.address)`
- Returns `{ digitalAsset: data ?? null, ...rest }`

**Hook 2: `useDigitalAssets(params: UseDigitalAssetsParams = {})`**

- `queryKey: digitalAssetKeys.list(filter, sort, limit, offset, include)`
- `queryFn: () => fetchDigitalAssets(url, { filter, sort, limit, offset, include })`
- Returns `{ digitalAssets: data?.digitalAssets ?? [], totalCount: data?.totalCount ?? 0, ...rest }`

**Hook 3: `useInfiniteDigitalAssets(params: UseInfiniteDigitalAssetsParams = {})`**

- Uses `useInfiniteQuery` with offset pagination
- `queryKey: digitalAssetKeys.infinite(filter, sort, include)` (SEPARATE namespace from list)
- Destructure `hasNextPage`, `fetchNextPage`, `isFetchingNextPage` BEFORE rest spread (avoids TS2783 per project decision)
- Flatten `data.pages` into single `digitalAssets` array using useMemo
- Returns `{ digitalAssets, hasNextPage, fetchNextPage, isFetchingNextPage, ...rest }`
- `getNextPageParam`: return `lastPageParam + pageSize` if `lastPage.digitalAssets.length >= pageSize`, otherwise `undefined`

**File 2: Create `packages/next/src/actions/digital-assets.ts`**

Follow the EXACT pattern from `packages/next/src/actions/profiles.ts`.

```typescript
'use server';

import type { FetchDigitalAssetsResult } from '@lsp-indexer/node';
import { fetchDigitalAsset, fetchDigitalAssets, getServerUrl } from '@lsp-indexer/node';
import type {
  DigitalAsset,
  DigitalAssetFilter,
  DigitalAssetInclude,
  DigitalAssetSort,
} from '@lsp-indexer/types';

export async function getDigitalAsset(
  address: string,
  include?: DigitalAssetInclude,
): Promise<DigitalAsset | null> {
  const url = getServerUrl();
  return fetchDigitalAsset(url, { address, include });
}

export async function getDigitalAssets(params?: {
  filter?: DigitalAssetFilter;
  sort?: DigitalAssetSort;
  limit?: number;
  offset?: number;
  include?: DigitalAssetInclude;
}): Promise<FetchDigitalAssetsResult> {
  const url = getServerUrl();
  return fetchDigitalAssets(url, params ?? {});
}
```
