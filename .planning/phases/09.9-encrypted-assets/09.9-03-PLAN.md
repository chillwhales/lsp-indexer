**Hook 1: `useEncryptedAssets`** â€” paginated list.

Return type alias: `UseEncryptedAssetsReturn<F>`:

```typescript
type UseEncryptedAssetsReturn<F> = Omit<
  UseQueryResult<{ encryptedAssets: F[]; totalCount: number }, Error>,
  'data'
> & {
  encryptedAssets: F[];
  totalCount: number;
};
```

**NOTE:** The destructured property name is `encryptedAssets` (matching `FetchEncryptedAssetsResult.encryptedAssets`).

3 overloads:

1. No include â†’ `UseEncryptedAssetsReturn<EncryptedAsset>`
2. `<const I>` with include â†’ `UseEncryptedAssetsReturn<EncryptedAssetResult<I>>`
3. Optional include â†’ `UseEncryptedAssetsReturn<PartialEncryptedAsset>`

Implementation:

```typescript
const { data, ...rest } = useQuery({
  queryKey: encryptedAssetKeys.list(
    params?.filter,
    params?.sort,
    params?.limit,
    params?.offset,
    params?.include,
  ),
  queryFn: () =>
    fetchEncryptedAssets({
      url: getClientUrl(),
      ...params,
    }),
});

return {
  encryptedAssets: data?.encryptedAssets ?? [],
  totalCount: data?.totalCount ?? 0,
  ...rest,
};
```
