The `lsp12_issued_asset` Hasura table (schema.graphql lines 3058–3073):

```
address: String!                          → issuerAddress (who issued the asset)
array_index: numeric                      → arrayIndex (position in LSP12 array, nullable)
asset_address: String!                    → assetAddress (the digital asset address)
id: String!                              → EXCLUDED (opaque Hasura ID)
interface_id: String                     → interfaceId (ERC165 interface, nullable)
issuedAsset: digital_asset               → digitalAsset relation
issued_asset_id: String                  → EXCLUDED (FK only)
timestamp: timestamptz!                  → timestamp (when indexed)
universalProfile: universal_profile      → issuerProfile relation
universal_profile_id: String             → EXCLUDED (FK only)
```

**IssuedAssetSchema** — same structure as CreatorSchema but different field names:

```typescript
import { z } from 'zod';
import { SortDirectionSchema, SortNullsSchema } from './common';
import type { IncludeResult, PartialExcept } from './include-types';
import {
  DigitalAssetIncludeSchema,
  DigitalAssetSchema,
  type DigitalAssetInclude,
  type DigitalAssetResult,
} from './digital-assets';
import {
  ProfileIncludeSchema,
  ProfileSchema,
  type ProfileInclude,
  type ProfileResult,
} from './profiles';

export const IssuedAssetSchema = z.object({
  /** Address that issued the digital asset (always present) */
  issuerAddress: z.string(),
  /** Digital asset contract address (always present) */
  assetAddress: z.string(),
  /** Position in the LSP12 issued assets array (null = not included or not set) */
  arrayIndex: z.number().nullable(),
  /** ERC165 interface ID (null = not included or not set) */
  interfaceId: z.string().nullable(),
  /** Timestamp when indexed (ISO string) */
  timestamp: z.string(),
  /** Universal Profile of the issuer (null = not included in query) */
  issuerProfile: ProfileSchema.nullable(),
  /** Digital asset details (null = not included in query) */
  digitalAsset: DigitalAssetSchema.nullable(),
});
```

**Conditional include types (DX-04)** — same pattern as Creator with `ResolveIssuerProfile` and `ResolveIssuedAssetDigitalAsset`:

```typescript
type IssuedAssetScalarIncludeFieldMap = {
  arrayIndex: 'arrayIndex';
  interfaceId: 'interfaceId';
  timestamp: 'timestamp';
};

type ResolveIssuerProfile<I> = I extends { issuerProfile: infer P }
  ? P extends ProfileInclude
    ? { issuerProfile: ProfileResult<P> | null }
    : {}
  : {};

type ResolveIssuedAssetDigitalAsset<I> = I extends { digitalAsset: infer D }
  ? D extends DigitalAssetInclude
    ? { digitalAsset: DigitalAssetResult<D> | null }
    : {}
  : {};

export type IssuedAssetResult<I extends IssuedAssetInclude | undefined = undefined> =
  I extends undefined
    ? IssuedAsset
    : IncludeResult<
        IssuedAsset,
        'issuerAddress' | 'assetAddress',
        IssuedAssetScalarIncludeFieldMap,
        I
      > &
        ResolveIssuerProfile<NonNullable<I>> &
        ResolveIssuedAssetDigitalAsset<NonNullable<I>>;

export type PartialIssuedAsset = PartialExcept<IssuedAsset, 'issuerAddress' | 'assetAddress'>;
```

```graphql
query GetIssuedAssets(...) {
  lsp12_issued_asset(where: $where, order_by: $order_by, limit: $limit, offset: $offset) {
    # Base fields — always fetched
    address          # This is issuerAddress (NOT assetAddress!)
    asset_address    # This is assetAddress

    # Include-toggled scalars
    array_index @include(if: $includeArrayIndex)
    interface_id @include(if: $includeInterfaceId)
    timestamp @include(if: $includeTimestamp)

    # Issuer's Universal Profile — Hasura field name is "universalProfile" (NOT "issuerProfile")
    universalProfile @include(if: $includeIssuerProfile) {
      address
      lsp3Profile {
        ... (COPY profile sub-field selections from GetCreatorsDocument's creatorProfile block,
             but replace $includeCreatorProfile* with $includeIssuerProfile*)
      }
      followedBy_aggregate @include(if: $includeIssuerProfileFollowerCount) { aggregate { count } }
      following_aggregate @include(if: $includeIssuerProfileFollowingCount) { aggregate { count } }
    }

    # Digital Asset — Hasura field name is "issuedAsset" (NOT "digitalAsset")
    issuedAsset @include(if: $includeDigitalAsset) {
      ... (COPY DA sub-field selections from GetCreatorsDocument's digitalAsset block — identical)
    }
  }

  lsp12_issued_asset_aggregate(where: $where) {
    aggregate { count }
  }
}
```

- `issuedAsset` (Hasura) → `digitalAsset` (our type) — the parser does this mapping
