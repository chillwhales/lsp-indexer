**Create `packages/types/src/owned-assets.ts`** with ALL OwnedAsset domain Zod schemas, following the same structure as `digital-assets.ts` and `nfts.ts`.

An OwnedAsset represents LSP7 fungible token ownership — how much of a token an address holds.

**Hasura `owned_asset` table fields:**

- `address: String!` — asset contract address
- `balance: numeric!` — owned balance (uint256 as Hasura numeric)
- `block: Int!` — block number
- `digitalAsset: digital_asset` — relation to digital_asset
- `digital_asset_id: String` — FK to digital_asset
- `id: String!` — unique ID
- `owner: String!` — owner address
- `timestamp: timestamptz!` — ISO timestamp
- `tokenIds: [owned_token!]!` — array relation to owned_tokens
- `tokenIds_aggregate: owned_token_aggregate!` — aggregate
- `universalProfile: universal_profile` — relation to universal_profile
- `universal_profile_id: String` — FK

**OwnedAssetSchema:**

```typescript
import { z } from 'zod';
import { SortDirectionSchema, SortNullsSchema } from './common';
import { DigitalAssetSchema, DigitalAssetIncludeSchema } from './digital-assets';
import { ProfileSchema } from './profiles';

export const OwnedAssetSchema = z.object({
  /** Unique identifier */
  id: z.string(),
  /** Asset contract address */
  address: z.string(),
  /** Owner address */
  owner: z.string(),
  /** Token balance — bigint for uint256 precision (consumer formats) */
  balance: z.bigint(),
  /** Block number when this ownership was last updated */
  block: z.number(),
  /** Timestamp when this ownership was last updated (ISO string) */
  timestamp: z.string(),
  /** Related digital asset details (null = not included in query) */
  digitalAsset: DigitalAssetSchema.nullable(),
  /** Related universal profile details (null = not included in query) */
  universalProfile: ProfileSchema.nullable(),
  /** Count of individual tokens (LSP8 token IDs) within this ownership, null if not included */
  tokenIdCount: z.number().nullable(),
});
```

**CRITICAL:** `balance` uses `z.bigint()` — per locked decision, the parser returns `bigint` and the consumer handles formatting.

**OwnedAssetFilterSchema** — all filterable fields from CONTEXT.md:

```typescript
export const OwnedAssetFilterSchema = z.object({
  /** Case-insensitive match on owner address */
  owner: z.string().optional(),
  /** Case-insensitive match on asset contract address */
  address: z.string().optional(),
  /** Case-insensitive match on digital_asset_id FK */
  digitalAssetId: z.string().optional(),
  /** Case-insensitive match on universal_profile_id FK */
  universalProfileId: z.string().optional(),
});
```

Note: `balance` and `timestamp` are not string-based filters but comparison-based. For the MVP, expose the string FK/address fields that match the established `_ilike` pattern. If balance/timestamp range filters are needed, they can be added later.

**OwnedAssetSortFieldSchema** — all direct columns + nested sorts from CONTEXT.md:

```typescript
export const OwnedAssetSortFieldSchema = z.enum([
  'balance',
  'timestamp',
  'address',
  'owner',
  'block',
]);
```

Note: Nested sorts (digitalAsset name, universalProfile, tokenIds_aggregate count) are supported at the service layer by mapping field names to nested Hasura order_by objects. The enum covers direct column sorts. For nested sorts, use separate enum values like `'digitalAssetName'` and `'tokenIdCount'`:

```typescript
export const OwnedAssetSortFieldSchema = z.enum([
  'balance',
  'timestamp',
  'address',
  'owner',
  'block',
  'digitalAssetName',
  'tokenIdCount',
]);
```

**OwnedAssetSortSchema, OwnedAssetIncludeSchema:**

```typescript
export const OwnedAssetSortSchema = z.object({
  field: OwnedAssetSortFieldSchema,
  direction: SortDirectionSchema,
  nulls: SortNullsSchema.optional(),
});

export const OwnedAssetIncludeSchema = z.object({
  /** Include related digital asset details — sub-fields control which DA attributes to fetch */
  digitalAsset: DigitalAssetIncludeSchema.optional(),
  /** Include related universal profile details */
  universalProfile: z.boolean().optional(),
  /** Include count of individual token IDs (tokenIds_aggregate) */
  tokenIdCount: z.boolean().optional(),
});
```

**Hook parameter schemas:** `UseOwnedAssetParamsSchema` (single by id), `UseOwnedAssetsParamsSchema` (list), `UseInfiniteOwnedAssetsParamsSchema` (infinite). Follow exact same pattern as digital assets:

```typescript
export const UseOwnedAssetParamsSchema = z.object({
  /** Owned asset unique ID */
  id: z.string(),
  /** Control which nested data to include */
  include: OwnedAssetIncludeSchema.optional(),
});

export const UseOwnedAssetsParamsSchema = z.object({
  filter: OwnedAssetFilterSchema.optional(),
  sort: OwnedAssetSortSchema.optional(),
  limit: z.number().optional(),
  offset: z.number().optional(),
  include: OwnedAssetIncludeSchema.optional(),
});

export const UseInfiniteOwnedAssetsParamsSchema = z.object({
  filter: OwnedAssetFilterSchema.optional(),
  sort: OwnedAssetSortSchema.optional(),
  pageSize: z.number().optional(),
  include: OwnedAssetIncludeSchema.optional(),
});
```

Export all inferred types at bottom.

An OwnedToken represents LSP8 individual NFT ownership — which specific token an address holds.

**Hasura `owned_token` table fields:**

- `address: String!` — asset contract address
- `block: Int!` — block number
- `digitalAsset: digital_asset` — relation
- `digital_asset_id: String` — FK
- `id: String!` — unique ID
- `nft: nft` — relation to nft
- `nft_id: String` — FK
- `ownedAsset: owned_asset` — relation
- `owned_asset_id: String` — FK
- `owner: String!` — owner address
- `timestamp: timestamptz!` — ISO timestamp
- `token_id: String!` — specific token ID
- `universalProfile: universal_profile` — relation
- `universal_profile_id: String` — FK

**OwnedTokenSchema:**

```typescript
export const OwnedTokenSchema = z.object({
  id: z.string(),
  address: z.string(),
  owner: z.string(),
  tokenId: z.string(),
  block: z.number(),
  timestamp: z.string(),
  /** Related digital asset (null = not included) */

  nft: NftSchema.nullable(),
  /** Related owned asset (parent fungible ownership record, null = not included) */
  ownedAsset: OwnedAssetSchema.nullable(),
  /** Related universal profile (null = not included) */

});
```
