**Create `packages/react/src/hooks/owned-assets.ts`** — 3 hooks following exact pattern from `hooks/nfts.ts` and `hooks/digital-assets.ts`:

1. **`useOwnedAsset(params: UseOwnedAssetParams)`** — single owned asset by ID

   - Uses `useQuery` with `ownedAssetKeys.detail(id, include)` as query key
   - `queryFn: () => fetchOwnedAsset(url, params)`
   - `enabled: Boolean(params.id)` — disabled when ID is falsy
   - Returns `{ ownedAsset: data ?? null, ...rest }`

2. **`useOwnedAssets(params: UseOwnedAssetsParams = {})`** — paginated list

   - Uses `useQuery` with `ownedAssetKeys.list(filter, sort, limit, offset, include)`
   - `queryFn: () => fetchOwnedAssets(url, params)`
   - Returns `{ ownedAssets: data?.ownedAssets ?? [], totalCount: data?.totalCount ?? 0, ...rest }`

3. **`useInfiniteOwnedAssets(params: UseInfiniteOwnedAssetsParams = {})`** — infinite scroll
   - Uses `useInfiniteQuery` with `ownedAssetKeys.infinite(filter, sort, include)`
   - `pageSize` default 20, offset-based pagination
   - `getNextPageParam`: if last page has fewer results than pageSize → undefined (no more pages), else `lastPageParam + pageSize`
   - **Destructure infinite query properties before rest spread** (TS2783 pattern from NFTs)
   - Returns `{ ownedAssets, hasNextPage, fetchNextPage, isFetchingNextPage, ...rest }`
   - `useMemo` to flatten pages into single array

Import `getClientUrl` from `@lsp-indexer/node` for URL resolution. Add JSDoc with @example for each hook.

**Create `packages/next/src/actions/owned-assets.ts`** — 2 server actions:

```typescript
'use server';

import { fetchOwnedAsset, fetchOwnedAssets, getServerUrl } from '@lsp-indexer/node';
import type { UseOwnedAssetParams, UseOwnedAssetsParams } from '@lsp-indexer/types';

export async function getOwnedAsset(params: UseOwnedAssetParams) {
  const url = getServerUrl();
  return fetchOwnedAsset(url, params);
}

export async function getOwnedAssets(params: UseOwnedAssetsParams) {
  const url = getServerUrl();
  return fetchOwnedAssets(url, params);
}
```
