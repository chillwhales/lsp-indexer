    **If extractable utilities were found:**

    1. Fork `chillwhales/LSPs` (if not already forked):
       ```bash
       gh repo fork chillwhales/LSPs --clone=false 2>/dev/null || echo "Fork exists"
       ```

    2. Clone the fork:
       ```bash
       gh repo clone {user}/LSPs /tmp/chillwhales-lsps-pr 2>/dev/null
       cd /tmp/chillwhales-lsps-pr
       ```

    3. Create a branch:
       ```bash
       git checkout -b feat/add-{utility-name}
       ```

    4. Add the utility to the appropriate package (e.g., `packages/utils/src/`):
       - Adapt the function signature to match the upstream coding style
       - Add tests if the upstream repo has a test setup
       - Update the package's index.ts exports

    5. Open the PR:
       ```bash
       gh pr create --repo chillwhales/LSPs --title "feat({pkg}): add {utility}" --body "..."
       ```

    6. Record the PR URL in the audit document.

    **If NO extractable utilities were found (all local utilities are too indexer-specific):**
