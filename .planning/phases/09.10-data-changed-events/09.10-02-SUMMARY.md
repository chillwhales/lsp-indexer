## Decisions Made

- **@lsp-indexer/data-keys package:** Standalone read-only registry with 32 hardcoded built-in ERC725Y data keys, no @lukso/\* dependencies
- **Prefix fallback matching:** Array keys (16 bytes), mapping keys (12 bytes), and explicit prefixes (6 bytes) matched via longest-prefix-first fallback
- **Module-load-time map:** Small map (~32 entries) built once â€” no lazy init needed, negligible startup cost
- **NFT name filter dual source:** `_or` pattern for `lsp4Metadata.name` + `lsp4MetadataBaseUri.name`, matching the established nfts service pattern
- **Profile prefix replacement:** `includeProfile*` â†’ `includeUniversalProfile*` for UP sub-includes (matching encrypted-assets pattern)
- **DA prefix replacement:** `include*` â†’ `includeDigitalAsset*` for DA sub-includes (matching issued-assets pattern)
